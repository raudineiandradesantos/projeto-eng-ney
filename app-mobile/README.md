# App Mobile - Dashboard do Proprietário

## Descrição

Este módulo contém o aplicativo Android (APK) desenvolvido para o **dono da empresa** acompanhar os dados gerados pelo site principal (projeto-eng-ney).

O app **não substitui o site** — ele trabalha em conjunto, consumindo os dados gerados pelo sistema web e exibindo relatórios e informações resumidas de forma prática no celular.

## Funcionalidades Previstas

- Visualização de dados e métricas geradas pelo site
- Dashboard com resumo de indicadores
- Notificações de novos registros
- Acesso rápido e seguro via autenticação

## Tecnologias

- Android (APK nativo ou WebView)
- Integração com a API / backend do site principal
